# Pre Security Path Notes

## Overview
The Pre Security path introduced the fundamental concepts of IT, networking, operating systems and cybersecurity.

The goal of this path was to build foundational knowledge before moving into more advanced security topics.

---

# Networking Fundamentals

## Topics Covered
- What is Networking
- Intro to LAN
- OSI Model
- Packets & Frames
- Extending Your Network
- DNS in Detail
- HTTP in Detail
- How Websites Work

## Key Things Learned
- Devices communicate through networks using protocols.
- The OSI model explains how network communication works in layers.
- DNS translates domain names into IP addresses.
- HTTP/HTTPS are used for website communication.
- Packets are units of transmitted data.

## Important Terms
- IP Address
- MAC Address
- DNS
- Router
- Switch
- HTTP
- HTTPS
